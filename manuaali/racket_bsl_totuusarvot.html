2.3 Totuusarvot

procedure

(boolean? x) → totuusarvo

  x : ei-tyyppivaatimusta
Tutkii, onko x tyyppiä totuusarvo (#true tai #false)
> (boolean? 42)

#false

> (boolean? #false)

#true

procedure

(boolean=? x y) → totuusarvo

  x : totuusarvo
  y : totuusarvo
Tutkii ovatko kaksi totuusarvoa samat.
> (boolean=? #true #false)

#false

> (boolean=? #false #false)

#true

procedure

(false? x) → totuusarvo

  x : ei-tyyppivaatimusta
Tutkii onko x tyyppiä totuusarvo ja sen arvo #false.
> (false? #false)

#true

procedure

(not x) → totuusarvo

  x : totuusarvo
Kääntää totuusarvon päinvastaiseksi.
> (not #false)

#true

value

true : totuusarvo

Vaihtoehtoinen tapa ilmaista #true ja #t. WeScheme palauttaa totuusarvot aina tässä muodossa (ilman # merkkiä).

> true

#true

value

false : totuusarvo

Vaihtoehtoinen tapa ilmaista #false ja #f. WeScheme palauttaa totuusarvot aina tässä muodossa (ilman # merkkiä).

> false

#false
